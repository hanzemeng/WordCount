# WordCount
Counts the frequency of all words in a txt file.

HOW TO USE

1. Download x64 folder and open it
2. Put txt file(s) into the RawFile folder
3. Run CodePlayGround.exe
4. Enter the name of the txt file
5. Processed file(s) can be found in the ProcessedFile folder

CHECK SOURCE CODE

1. Download CodePlayground folder and CodePlayground.sln
2. Open CodePlayground.sln with Visual Studio
